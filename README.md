# Iran Gilaki Co Merch Site

Separate workspace for building a standalone merch landing page for the user's wife.

## Goal
Create a mobile-first, desktop-friendly static website similar in spirit to https://www.germanshepherd.love that showcases 16 Amazon Merch listings for the brand "Iran Gilaki Co".

## Separation rule
This project is intentionally unrelated to Blackjack Bailey VR and should remain isolated in content, assets, planning, and implementation.

## Deliverables
- `index.html` — ready-to-host single-page storefront
- marketplace links per item for `USA`, `UK`, and `DE`
- product thumbnail/title opens the USA listing

## Marketplace link behavior
- The main product image/title links to the USA listing.
- The `USA`, `UK`, and `DE` links below each product open the corresponding Amazon marketplace.
- Where an exact UK or DE product page could not be confidently confirmed, the page uses a marketplace-specific Amazon search query for that product title plus the `Iran Gilaki Co` brand name.
